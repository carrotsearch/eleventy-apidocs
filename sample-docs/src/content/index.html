Heads up

Callouts use plain <div class="info"> or warning markup — no extra plugin syntax needed.

Why HTML, not Markdown?

Because HTML is the lingua franca of the web and lets us express anything the browser can render — without fighting flavor mismatches or escape hatches.

Want the canonical reference for callouts and code blocks? The configuration section of Getting started exercises both.
